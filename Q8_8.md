U8.8 is a 16-bit unsigned fixed-point format with 8 integer bits and 8 fractional bits. It represents non‑negative real numbers with 1/256 resolution.

Key properties
- Bit layout: 16 bits total = 8 integer bits (most-significant) + 8 fractional bits (least-significant).
- Scaling: interpret the 16-bit unsigned integer N as the value N / 2^8.
- Range:
  - Minimum = 0.0 (0x0000).
  - Maximum = (2^16 − 1) / 2^8 = 65535 / 256 = 255.99609375 (0xFFFF).
- Resolution (LSB): 2^(−8) = 1/256 ≈ 0.00390625.

Representation examples
- 0x0100 (256) → 256 / 256 = 1.0
- 0x0080 (128) → 128 / 256 = 0.5
- 0x00FF (255) → 255 / 256 ≈ 0.99609375
- 0xFF00 (65280) → 65280 / 256 = 255.0

Conversions
- To fixed (from unsigned integer x): u8_8 = x << 8.
- From fixed to integer (floor): int = u8_8 >> 8.
- From float f (0 ≤ f ≤ 255.99609375): u8_8 = (uint16_t)roundf(f * 256.0f), then clamp to 0..65535.
- To float: f = u8_8 / 256.0f.

Arithmetic notes
- Addition/subtraction: operate on 16-bit unsigned integers; watch for overflow (wraps). If saturation desired, clamp to 0..65535.
- Multiplication: (uint32_t)A * B gives a 32-bit product scaled by 2^16 (since each operand has 8 fractional bits → product has 16 fractional bits). To get back to U8.8, shift right by 8:
  - product32 = (uint32_t)A * (uint32_t)B; result = (product32 + rounding) >> 8.
- Divide: promote to wider integer, shift left by 8 before dividing to preserve fraction: result = (A << 8) / B (handle B=0).
- Rounding: add half LSB (1 << 7) before right shift for nearest rounding.
- Overflow care: intermediate ops should use 32-bit math to avoid losing precision.

Use cases
- Embedded systems and graphics where non-negative quantities with modest integer range (0–255) and 1/256 resolution are needed (e.g., gamma ramps, color channel scaling, small-scale DSP, meter readings).
- Faster and smaller than floating point on constrained hardware; deterministic fixed resolution.

Compact C examples
```c
// Convert float to U8.8 (clamped)
uint16_t float_to_u8_8(float f){
    int32_t v = (int32_t)roundf(f * 256.0f);
    if (v < 0) return 0;
    if (v > 0xFFFF) return 0xFFFF;
    return (uint16_t)v;
}

// Multiply two U8.8 with rounding, result saturated to U8.8
uint16_t u8_8_mul(uint16_t a, uint16_t b){
    uint32_t prod = (uint32_t)a * (uint32_t)b;      // Q16 fractional
    uint32_t rounded = (prod + (1u << 7)) >> 8;    // back to U8.8
    if (rounded > 0xFFFF) return 0xFFFF;
    return (uint16_t)rounded;
}
```
