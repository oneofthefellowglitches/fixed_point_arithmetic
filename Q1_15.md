## Q15 (Q1.15) — 16-bit signed fixed-point

Q15 (often written Q1.15) is a fixed-point numeric format that represents signed numbers using 16 bits with 15 fractional bits and 1 integer/sign bit. Key properties:

- Bit layout (16 bits total): 1 sign/integer bit + 15 fractional bits.
- Value range: from −1.0 (inclusive) to just under +1.0 (exclusive). Specifically:
  - Minimum representable value = −1.0 (binary 1000 0000 0000 0000 = −1.0 in two’s complement).
  - Maximum representable value = (1 − 2^(−15)) = 0.999969482421875 (binary 0111 1111 1111 1111).
- Resolution (LSB): 2^(−15) = 1/32768 ≈ 3.0517578125e−5.
- Representation: interpret the 16-bit two’s-complement integer N as the real number N / 2^15.
  - Example: 0x4000 (decimal 16384) → 16384 / 32768 = 0.5.
  - Example: 0x8000 (decimal −32768) → −32768 / 32768 = −1.0.
  - Example: 0x7FFF (decimal 32767) → 32767 / 32768 ≈ 0.99996948.
- Arithmetic notes:
  - Addition/subtraction: use 16-bit two’s-complement addition; overflow wraps unless you detect/saturate. Resulting fixed-point value still scaled by 2^15.
  - Multiplication: multiplying two Q15 values yields a Q30 result (scaled by 2^30). To return to Q15, shift right by 15 bits (and handle sign and rounding). Typical sequence: product = (int32)a * (int32)b; result_q15 = (product + rounding) >> 15.
  - Scaling and shifts: left shift increases integer magnitude (may overflow), right shift divides by powers of two (preserve sign with arithmetic shift).
  - Conversion to/from integers:
    - To convert int x to Q15: x_q15 = x << 15.
    - To convert Q15 to int (floor): x_int = x_q15 >> 15 (arithmetic shift).
  - Conversion to/from floating point:
    - float = fixed_q15 / 32768.0f
    - fixed_q15 = (int16)roundf(float * 32768.0f), but clamp to [−32768, 32767].
- Common uses: audio signal processing (samples in ±1.0 range), DSP algorithms, embedded systems without FPU.
- Advantages: predictable, fast integer math, compact (16 bits). Limitations: limited dynamic range (±1), care required with scaling to avoid overflow and quantization noise.
